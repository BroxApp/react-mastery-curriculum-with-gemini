# Mini Project: Responsive Landing Page

## 🎯 Goal
Build a modern, responsive landing page using CSS fundamentals.

## 🏗️ Requirements (The "Big 5")
- **Flexbox**: Layout for Navbar, Features, and Footer.
- **CSS Grid**: Hero section or Photo Gallery.
- **Responsive Design**: Fluid layouts and design principles.
- **Media Queries**: Breakpoints for Desktop, Tablet, and Mobile.
- **Animations**: Entrance effects or hover transitions.

## 🧱 Architecture Plan
1. **HTML structure**: Semantically organized.
2. **Global CSS**: Variables (colors, spacing).
3. **Sections**: 
   - Navbar (Flexbox)
   - Hero Section (Grid + Animation)
   - Features Section (Responsive Flex/Grid)
   - Footer (Flexbox)
