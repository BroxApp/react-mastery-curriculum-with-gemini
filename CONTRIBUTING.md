# Contributing to React Mastery Project

Hi! I'm Brox. This project follows a strict engineering workflow.

## How to contribute:

1. **Fork** the repository.
2. **Create a new branch** using `git checkout -b feature/your-feature-name`.
3. **Commit your changes** with a descriptive message (e.g., `feat: add new learning module`).
4. **Push to the branch** and open a **Pull Request**.
5. Wait for the mentor's review!
